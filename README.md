# I@A

IIROSE `@全体成员` 网页端插件。

## 版本

- `v0.1.1`

## 功能

- 在公屏发送 `[@全体成员]` 时，自动展开当前房间内可提及成员。
- 自动排除自己、重复用户名、空用户名和非法换行用户名。
- `v0.1.1` 起会排除房间内人工智能/机器人对象，不再把它们加入 `@` 列表。

## AI 排除策略

插件会组合使用以下信息进行过滤：

- IIROSE 前端运行时提供的 `Constant.BOT_UID`、`Constant.BOT_NAME`、`Constant.BOT` 名单。
- 房间成员原始数据中的稳定类型位和扩展标记。
- 成员简介或对象字段中明显的 `AI`、`Bot`、`人工智能`、`机器人`、`智能体` 等文本特征。

策略偏保守，优先避免把明显的 AI 对象加入 `@全体成员`。
